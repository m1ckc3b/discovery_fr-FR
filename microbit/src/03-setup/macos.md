# macOS

Tous les outils peuvent être installés en utilisant [Homebrew] :

[Homebrew] : http://brew.sh/

``` console
$ # Débogueur ARM GCC
$ brew install arm-none-eabi-gdb

$ # Minicom
$ brew install minicom

$ # lsusb (liste des périphériques USB connectés)
$ brew install lsusb
```
